## Sequence Diagram: Medicine and Painkiller Status Overview

### Presentation → Application
```mermaid
sequenceDiagram
    actor TimeEvent
    participant MedicineStatusPage as Presentation Layer
    participant MedicineStatusHandler as Application Layer
    TimeEvent ->> MedicineStatusPage: RequestMedicineStatus(shift)
    MedicineStatusPage ->> MedicineStatusHandler: GetMedicineStatus(shiftDto)
    MedicineStatusHandler -->> MedicineStatusPage: MedicineStatusDto
    alt ErrorLoadingMedicineData
        MedicineStatusHandler -->> MedicineStatusPage: Error(message)
    else NoMedicineHistoryFound
        MedicineStatusHandler -->> MedicineStatusPage: NoRecords()
    else Success
        MedicineStatusHandler -->> MedicineStatusPage: MedicineStatusDto(residents, timestamps, given)
    end
    TimeEvent ->> MedicineStatusPage: RequestPainkillerStatus(last24h)
    MedicineStatusPage ->> MedicineStatusHandler: GetPainkillerStatus(last24hDto)
    MedicineStatusHandler -->> MedicineStatusPage: PainkillerStatusDto
    alt ErrorLoadingPainkillerData
        MedicineStatusHandler -->> MedicineStatusPage: Error(message)
    else NoPainkillerHistoryFound
        MedicineStatusHandler -->> MedicineStatusPage: NoRecords()
    else Success
        MedicineStatusHandler -->> MedicineStatusPage: PainkillerStatusDto(types, nextAllowedTime, residents)
    end
```

### Application → Infrastructure
```mermaid
sequenceDiagram
    participant MedicineStatusHandler as Application Layer
    participant MedicineAdministrationManager as Infrastructure Layer
    participant PainkillerAdministrationManager as Infrastructure Layer
    MedicineStatusHandler ->> MedicineAdministrationManager: FetchMedicineStatus(shiftDto)
    MedicineAdministrationManager -->> MedicineStatusHandler: MedicineStatusDto
    MedicineStatusHandler ->> PainkillerAdministrationManager: FetchPainkillerStatus(last24hDto)
    PainkillerAdministrationManager -->> MedicineStatusHandler: PainkillerStatusDto
```
