## System Sequence Diagram
```mermaid
sequenceDiagram
    actor TimeEvent
    participant System
    TimeEvent ->> System: requestMedicineSchedule(shift)
    alt errorLoadingMedicineData
        System -->> TimeEvent: displayErrorAndRetryOption()
    else noMedicineHistoryFound
        System -->> TimeEvent: displayNoRecordsMessage()
    else painkillerTypeNotRecognized
        System -->> TimeEvent: displayWarning()
    else success
        System -->> TimeEvent: returnMedicineAdministrationStatus(residents, timestamps, given)
    end
    TimeEvent ->> System: requestPainkillerSchedule(last24h)
    alt errorLoadingPainkillerData
        System -->> TimeEvent: displayErrorAndRetryOption()
    else noPainkillerHistoryFound
        System -->> TimeEvent: displayNoRecordsMessage()
    else painkillerTypeNotRecognized
        System -->> TimeEvent: displayWarning()
    else success
        System -->> TimeEvent: returnPainkillerAdministration(types, nextAllowedTime, residents)
    end
```
