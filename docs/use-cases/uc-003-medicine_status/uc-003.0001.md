## Use Case Description

## User story
As a time event, the dashboard displays medicine status for all residents in the last 24 hours, including the time medicine and painkillers were administered, so proper medication intervals and compliance can be monitored.

### Brief Use Case
**Title**: View Medicine Status on Dashboard (Last 24 Hours)
**Success Flow**:
The dashboard screen automatically shows medicine administration history for all residents in the last 24 hours, including timestamps and painkiller details with their respective types and next allowed timespan.

### Casual Use Case
**Title**: View Medicine Status on Dashboard (Last 24 Hours)
**Scope**: Dashboard Screen
**Level**: System
**Actors**:
- Time Event
**Main Flow**:
1: Dashboard screen updates
2: Dashboard displays medicine history for all residents in the last 24 hours with timestamps and a boolean indicator if it was given
3: Dashboard highlights painkiller entries, their types, and next allowed timespan for each
**Main Extensions**:
2a: No medicine history found for any resident in the last 24 hours
3a: Painkiller type not recognized

### Fully Dressed Use Case
**Title**: View Medicine Status on Dashboard (Last 24 Hours)
**Scope**: Dashboard Screen
**Level**: System
**Actors**:
- Time Event
**Related Requirements**:
- Medicine administration tracking
- Painkiller interval enforcement
- 24-hour filter logic
**Preconditions**:
- Dashboard screen is operational
- Medicine records exist for residents
**Main Flow**:
1. Dashboard screen is active
2: Dashboard displays medicine history for all residents in the last 24 hours with timestamps and a boolean indicator if it was given
3. Dashboard highlights painkiller entries, their types, and next allowed timespan for each
**Extensions**:
a: At any time, system fails to load medicine data  
    1: Display error message and retry option  
2a: No medicine history found for any resident in the last 24 hours  
    1: Display 'No medicine records found' message  
3a: Painkiller type not recognized  
    1: Display 'Unknown painkiller type' warning  

**Postconditions**:
- Dashboard displays medicine and painkiller status for all residents in the last 24 hours
- Dashboard informs about next allowed painkiller dose for each resident
