## Systemsekvensdiagram
```mermaid
sequenceDiagram
    actor TimeEvent
    participant System
    TimeEvent ->> System: anmodOmMedicinSkema(vagt)
    alt fejlVedIndlæsningAfMedicinData
        System -->> TimeEvent: visFejlOgPrøvIgenMulighed()
    else ingenMedicinHistorikFundet
        System -->> TimeEvent: visIngenPosterBesked()
    else smertestillendeTypeIkkeGenkendt
        System -->> TimeEvent: visAdvarsel()
    else succes
        System -->> TimeEvent: returnerMedicinAdministrationsStatus(beboere, tidsstempler, givet)
    end
    TimeEvent ->> System: anmodOmSmertestillendeSkema(sidste24t)
    alt fejlVedIndlæsningAfSmertestillendeData
        System -->> TimeEvent: visFejlOgPrøvIgenMulighed()
    else ingenSmertestillendeHistorikFundet
        System -->> TimeEvent: visIngenPosterBesked()
    else smertestillendeTypeIkkeGenkendt
        System -->> TimeEvent: visAdvarsel()
    else succes
        System -->> TimeEvent: returnerSmertestillendeAdministration(typer, næsteTilladteTidspunkt, beboere)
    end
```
