## Wireframe

Dashboard bottom area shows shared sections for the current shift.
PhoneList is the left-hand section in the bottom row, alongside "Særligt ansvar" and "Afdelingsopgaver".
PhoneList only displays which staff member carries each fixed phone number for the active shift; there is no call or contact functionality.

Overview layout (Dashboard — bottom row):
+--------------------------------------------------------------------------+
|  [Main dashboard content above — resident cards, tasks, etc.]            |
+--------------------------------------------------------------------------+
|  [Telefonnummer (PhoneList)] |  [Særligt ansvarlige / Navn] |  [Afdelingsopgaver]      |
|   (this wireframe)           (shared sections)               (shared sections)         |
+--------------------------------------------------------------------------+

Detail layout — PhoneList section:

- Phone numbers are fixed labels and cannot be added or removed.
- Staff name fields are editable for the current shift only.
- Shift selector [Shift: D/E/N] is a read-only indicator of the active system shift
  (Day, Evening, Night) driven by system time.

## Notes
- Layout follows the physical Excel overlap sheet used at Slottet.
- This is not a contact list — there is no click-to-call or dialing.
- Assignments apply per shift; staff update their name when they pick up a phone.
- List content refreshes automatically every 60 seconds based on a system time event.
- Phone numbers are shared resources; the same staff member may appear on multiple rows.
- PhoneList is visually grouped with "Særligt ansvar" and "Afdelingsopgaver" at the bottom of the Dashboard.
