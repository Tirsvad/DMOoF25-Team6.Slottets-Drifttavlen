## Wireframe

Dashboardens nederste område viser fælles sektioner for det aktuelle vagtskift.
PhoneList er den venstre sektion i nederste række sammen med "Særligt ansvar" og "Afdelingsopgaver".
PhoneList viser kun, hvilken medarbejder der bærer hvert faste telefonnummer i den aktuelle vagt; der er ingen opkalds- eller kontaktfunktionalitet.

Overordnet layout (Dashboard — nederste række):
+--------------------------------------------------------------------------+
|  [Hovedindhold på dashboardet ovenfor — beboerkort, opgaver osv.]        |
+--------------------------------------------------------------------------+
|  [Telefonnumre (PhoneList)] |  [Særligt ansvarlige / Navn] |  [Afdelingsopgaver] |
|   (denne wireframe)         (fælles sektion)               (fælles sektion)      |
+--------------------------------------------------------------------------+

Detaljeret layout — PhoneList sektion:
+--------------------------------------------------------------+
|  PhoneList                                      [Vagt: D/A/N]|
+-----------------+--------------------------------------------+
|  Telefonnummer  |  Tildelt medarbejdernavn                  |
+-----------------+--------------------------------------------+
|  41522          |  [                                    ]    |
|  41523          |  [                                    ]    |
|  41524          |  [                                    ]    |
|  41525          |  [                                    ]    |
|  41526          |  [                                    ]    |
|  41527          |  [                                    ]    |
|  41529          |  [                                    ]    |
+-----------------+--------------------------------------------+
|  Sidst opdateret: [TT:MM] (automatisk opdatering hvert 60. sekund) |
+--------------------------------------------------------------+

- Telefonnumre er faste labels og kan ikke tilføjes eller fjernes.
- Medarbejdernavne-felterne er redigerbare kun for den aktuelle vagt.
- Vagtindikatoren [Vagt: D/A/N] er en skrivebeskyttet indikator for den aktive systemvagt
  (Dag, Aften, Nat) styret af systemtid.

## Notes
- Layoutet følger det fysiske Excel-overlap-ark, der bruges på Slottet.
- Dette er ikke en kontaktliste — der er ingen klik-for-at-ringe eller opkaldsfunktion.
- Tildelinger gælder pr. vagt; medarbejdere opdaterer deres navn, når de tager en telefon.
- Listen opdateres automatisk hvert 60. sekund baseret på en systemtidshændelse.
- Telefonnumre er fælles ressourcer; den samme medarbejder kan forekomme på flere rækker.
- PhoneList er visuelt grupperet med "Særligt ansvar" og "Afdelingsopgaver" nederst på Dashboardet.
