## System Sequence Diagram

```mermaid
sequenceDiagram
    actor Employee
    participant System
    Employee ->> System: selectResident(residentId)
    System -->> Employee: showResidentNotes(residentNotes)
    Employee ->> System: addResidentNote(residentId, noteText)
    alt addResidentNote success
        System -->> Employee: confirmResidentNoteAdded()
    else addResidentNote error
        System -->> Employee: errorResidentNote("ResidentNote cannot be saved")
    end
    Employee ->> System: editResidentNote(residentId, residentNoteId, newText)
    alt editResidentNote success
        System -->> Employee: confirmResidentNoteEdited()
    else editResidentNote error
        System -->> Employee: errorResidentNote("ResidentNote cannot be saved")
    end
    Employee ->> System: deleteResidentNote(residentId, residentNoteId)
    alt deleteResidentNote success
        System -->> Employee: confirmResidentNoteDeleted()
    else deleteResidentNote error
        System -->> Employee: errorResidentNote("ResidentNote cannot be deleted")
    end
```
