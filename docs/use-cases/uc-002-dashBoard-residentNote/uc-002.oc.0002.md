## Operation Contract
### Select Resident
- **Preconditions**: Employee is logged into the system and has access to resident data.
- **Postconditions**: System displays notes for the selected resident.

### Add Resident Note
- **Preconditions**: Employee has selected a resident and entered note text.
- **Postconditions**: System confirms the note is added and visible in the resident's notes list.

### Edit Resident Note
- **Preconditions**: Employee has selected a resident and a specific note to edit.
- **Postconditions**: System confirms the note is updated and the changes are visible.

### Delete Resident Note
- **Preconditions**: Employee has selected a resident and a specific note to delete.
- **Postconditions**: System confirms the note is deleted and it is no longer visible in the resident's notes list.
