## System Sequence Diagram

```mermaid
sequenceDiagram
    actor Caregiver
    participant AuthenticationSystem as System
    Caregiver ->> System: openLoginPage()
    Caregiver ->> System: enterCredentials(username, password)
    System -->> Caregiver: showValidationMessage()
    %% 2a: missing credentials
    alt credentialsValid
        System -->> Caregiver: createSession()
        System -->> Caregiver: redirectToDashboard()
    else credentialsInvalid
        System -->> Caregiver: showLoginError()
    else accountLocked
        System -->> Caregiver: showAccountLocked()
    else serviceUnavailable
        System -->> Caregiver: showServiceError()
    end
```
