## Operation Contract
### Login
- **Preconditions**: Caregiver is on the login page and has entered username and password.

#### Extensions
- **2a. Username or password missing**: System shows validation message.
- **3a. Credentials invalid**: System shows login error.
- **Account locked**: System shows account locked message.
- **Service unavailable**: System shows service error.
